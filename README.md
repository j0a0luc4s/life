# life

Conway's game of life implementation in C with raylib

## Installing raylib dependencies

Refer to raylib wiki

## Building

```
git submodule init
git submodule update
cd raylib
mkdir build
cd build
cmake ..
make
cd ../..
make
```

## Running

```
make run
```
